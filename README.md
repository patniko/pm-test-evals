# Issue Triage Demo

This repository is used for a PM training exercise on evaluating AI agents.

## About

This is a sample SDK project with open issues that need triage.
An issue triage agent will label, prioritize, and respond to these issues.

## Languages

This project has components in:
- **TypeScript/Node.js** (`src/`)
- **Python** (`python/`)
- **Go** (`go/`)
